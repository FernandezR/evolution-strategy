# Evolution Strategy

Simple implementation of Evolution Strategy written in python3. Evolution Strategy (ES) is an optimization technique based on ideas of adaptation and evolution

## Usage

- Install Numpy

```bash
pip install numpy
```

- Tweak constants in `es.py`, fitness function, number of iterations etc.
- Run

```python
python es.py
```
